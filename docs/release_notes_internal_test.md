# LifeTracker 内部テスト版リリースノート (2025-11-04)

## 概要
内部テストトラック向けの最新ビルドです。高度な設定メニューの導線やアナリティクス画面を中心に改善し、データエクスポート体験とポモドーロ連携の安定性を高めました。

## 主な更新
- **高度な設定セクション**: 設定画面からダッシュボード拡張やアナリティクス詳細にすぐアクセスできる新しい導線を追加。
- **アナリティクス画面の改善**: グラフ表示の安定性向上、予測チップにテストタグを付与して UI テストの信頼性を向上。
- **データエクスポートの磨き込み**: CSV/PDF エクスポートの例外処理を見直し、外部ストレージ書き込み時の失敗ケースを減少。
- **ポモドーロ連携の安定化**: セッション開始/終了通知の遅延を解消し、通知チャンネルの初期化を簡素化。
- **軽微な調整**: Lint 指摘の解消、未使用リソースの削除、アプリ起動時のログ整備。

## 既知の課題
- targetSdk 35 対応に伴う最新の権限挙動は引き続き検証が必要です。
- Firebase Test Lab 等での外部デバイステストは未実施のため、互換性レポートを随時収集してください。

## テスト観点
- 設定 > 高度な機能 から各画面へ遷移し、戻る操作でクラッシュしないこと。
- アナリティクス画面で最新の JSONL データが反映され、日別推移・予測チップが正しく表示されること。
- ポモドーロタイマー開始時に通知が重複しないこと、終了通知から履歴に正しく記録されること。
- CSV/PDF エクスポートが外部ドキュメント選択で成功し、ファイル内容が期待通りであること。
