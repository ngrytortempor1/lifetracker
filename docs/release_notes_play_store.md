# LifeTracker リリースノート（Play ストア提出用）

## バージョン 1.0.0 — 2025-11-04

LifeTracker の初回公開ビルドです。日々の予定・タスク・習慣・ポモドーロを一つのワークスペースにまとめ、振り返りと分析までをシームレスに行えるよう設計しました。

### 主な機能
- **ダッシュボード**: 今日の予定・締切・完了タスク・習慣達成状況をカード形式で表示し、ワンタップで各画面へ遷移できます。
- **タスク & 習慣管理**: マイデイ／重要／計画済みなどのビュー切り替え、習慣の進捗チェック、クイック入力に対応。自然言語解析で期限やリマインダーも自動補完します。
- **ポモドーロタイマー**: タスク／習慣とリンクできる 25-5 分ベースのカスタムタイマー。セッション履歴はライフログに自動反映されます。
- **クイックログ**: 日中の気づきや出来事をタグ付きで素早く記録。後述の分析に活用されます。
- **エクスポート機能**: ダッシュボードのスナップショットを CSV / PDF で出力し、バックアップや共有に利用できます。

### 行動遷移グラフ（Analytics）
- タスク完了の連続パターンを解析し、時間帯ごとの予測可能性を散布図で表示。自動化されているルーチンと変動の大きい時間帯を見極められます。
- ローリング平均チャートで習慣・タスク完了数の推移を追跡。期間やウィンドウ幅、イベント種別を切り替え可能です。
- クイックログの頻出タグランキングを合わせて表示し、どのテーマ・気分が多いか把握できます。
- すべてのグラフはテキストレポートモードを備え、画面読み上げや資料への貼り付けにも対応しています。

### その他の改善
- 設定画面に「高度な機能」セクションを追加し、ダッシュボード拡張や分析画面への導線を強化しました。
- エクスポート処理や通知チャンネルの初期化を見直し、失敗時のハンドリングと起動パフォーマンスを改善しました。
- Lint 指摘の解消や未使用リソースの整理など、リリース品質に向けた足回りを整えました。
