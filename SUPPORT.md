# LifeTracker サポートプラン

LifeTracker は GNU AGPL v3.0 ライセンスで公開されており、誰でも自由に利用・改変できます。OSS としての活動を持続させるため、以下の有償サポートプランを提供します。

## 1. 導入支援パッケージ
- 要件ヒアリングとセットアップ支援 (2 時間)
- JSONL / SQLite ストレージの運用設計サポート
- プラグイン追加やブランドカスタマイズの設計レビュー

## 2. 優先サポート (サブスクリプション)
- 平日 24 時間以内のメール回答保証
- バグ報告・改善提案の優先対応
- リリース計画やマイグレーションの技術相談

## 3. マネージドホスティング / バックアップ
- アプリビルド、署名、配布ワークフローの自動化
- クラウドストレージへの暗号化バックアップ
- 監視と障害一次対応

## 相談窓口
プランの詳細や見積もりは `kirikirinokirin@gmail.com` までご連絡ください。利用シナリオに応じてカスタムプランの提案も可能です。

OSS 版で十分な場合は、Issue や Pull Request でのコラボレーションも歓迎しています。プロジェクトを気に入っていただけたら、GitHub でのスターやブログでの紹介もぜひお願いします。
